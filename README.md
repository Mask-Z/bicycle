##自行车租赁系统
- 数据库采用mysql
- 采用MVC三层架构模式设计主体框架
- 本项目主要运用了SpringMVC+Hibernate的框架进行开发
- 使用Intellij进行开发的,具体的请clone项目，使用idea直接打开查看即可；
